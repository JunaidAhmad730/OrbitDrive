# 🌌 OrbitDrive

A modern cloud storage management platform powered by Telegram.

## 🚀 Overview

OrbitDrive is a cloud storage manager that provides a beautiful interface for organizing, searching, and managing files stored in Telegram.

## ✨ Features

- 🔐 Telegram Authentication
- 📁 Virtual Folder System
- 📄 Smart File Categorization
- ⭐ Favorites
- 🔍 Fast Search
- 📊 Storage Analytics
- 🗑 Trash Management
- 🌙 Dark Mode
- 📱 Responsive UI

## 🛠 Tech Stack

### Frontend

- Next.js
- React
- TypeScript
- Tailwind CSS

### Backend

- Node.js
- Express
- TypeScript

### Database

- PostgreSQL
- Prisma ORM

### Storage

- Telegram MTProto API

### DevOps

- Docker
- GitHub Actions
